# Creative Agency - Landing Page

![Vista Previa del Proyecto](./Public/IMG/Previwe.png)

Este proyecto es una landing page moderna para una agencia creativa, desarrollada como parte de mi formación en **Conckerblock**. El diseño destaca por una estructura de Grid compleja y efectos de superposición (overlay) realizados puramente con CSS.

## 🚀 Tecnologías Utilizadas

* **HTML5:** Estructura semántica avanzada.
* **CSS3:** * CSS Grid para el layout principal.
    * Flexbox para alineación de componentes.
    * Funciones modernas: `clamp()` para tipografía responsiva y `calc()` para manejo de desbordes controlados.
* **Vite:** Herramienta de construcción (Build tool).

## 🛠️ Desafíos Técnicos Resueltos

1.  **Layout Asimétrico:** Implementación de una columna derecha de fondo completo (`grid-row: 1/4`) que integra el botón de contacto y la imagen hero.
2.  **Efecto de Superposición:** Logré que la sección de categorías rompa el padding lateral y se superponga a la imagen derecha mediante el uso estratégico de márgenes negativos, `calc()` y contextos de apilamiento (`z-index`).
3.  **Optimización de Archivos:** Migración de assets a la carpeta `/public` para garantizar rutas relativas limpias y compatibilidad con despliegues en producción.
